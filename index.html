Cedar-OS

A Lite, Free, Open Source Operating System, Made From Scratch.

NOTE: This OS is in pre-release, it will improve over time.
WARNING: This OS may be unstable on real computers. Try to run in QEMU as we still have not tested it on real hardware.
GITHUB REPO DOWNLOAD IMG

How to Compile

Written in x86 assembly and C. Requires NASM.

# Run the build script
./build.sh

How to Run

Use QEMU to run the virtual machine. Turning it into an ISO for VMware or real hardware is not recommended yet.

Standard Run:

qemu-system-i386 -fda build/cedar.img

Debug:

qemu-system-i386 -fda build/cedar.img -d int -D fresh_test.txt -M smm=off -no-reboot -no-shutdown
